# 🏞️ 정주行 (Jungju-Heng)

청년 맞춤형 지역 정주 정책 추천·신청 연계 플랫폼  
2025년 제7회 K-디지털 트레이닝 해커톤 참가작 | **불만두 팀** 🚀

---

## ✨ 프로젝트 개요

**정주行**은 지방 청년이 복잡한 정책 정보를 **쉽게 탐색하고, AI 요약**으로 빠르게 이해하며,  
신청 절차까지 한 번에 안내받을 수 있도록 만든 **스마트 지역 정책 허브**입니다.  

---

## ⚙️ 주요 기능

- 🧩 **정밀 정책 매칭**  
  ‣ 나이, 거주지, 관심사 등 프로필 기반 청년 대상 정책 자동 추천

- 🤖 **AI 정책 요약**  
  ‣ OpenAI로 정책 설명 핵심만 간결하게 요약 제공

- 📝 **신청 가이드**  
  ‣ 필요 서류, 단계별 절차, 바로가기 등 신청 진입 장벽 해소

- 📱 **모바일 친화적 UI**  
  ‣ Vue 3 + Bootstrap 기반 반응형 웹 제공

---
